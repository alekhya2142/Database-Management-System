# Database Management System (DBMS) in C++

## 📌 Overview
This project is a simple **Database Management System** implemented in C++.  
It demonstrates how a DBMS works internally by allowing users to:
- Create tables with custom columns
- Insert records (rows) into tables
- Display all records in a table
- Search records by column values
- Manage multiple tables inside a single database

The system uses **in-memory storage** (vectors) and supports **basic indexing** for faster lookups.

---

## ⚙️ Features
- **Create Table**: Define a table schema with column names.  
- **Insert Record**: Add rows of data into a table.  
- **Display Table**: Show all records in a table with column names.  
- **Search Records**: Find records by column and value.  
- **Indexing**: Optionally index a column (e.g., `id`) for fast searches.  
- **Multiple Tables**: Manage more than one table in the same database.
